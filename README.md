Un programa que simule administrar un supermercado, con una estructura para los productos y una estructura para los empleados (y capaz una para socios).
El programa  tendría una pila de productos que llegan cada día(usando el tiempo real de la computadora y la librería time) y permitiría asignar cada producto
que llega a una sección del supermercado y un precio. Cada día se vende una cantidad al azar de productos y eso afecta al balance de plata del supermercado. 
Además se podría buscar los productos por nombre y se puede acceder a la lista de empleados para eliminar cuando se despide a uno y a agregar a alguien cuándo llega. 
Habría una opción para pedir productos para que lleguen al día siguiente
